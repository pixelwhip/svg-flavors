First and foremost a graphics format. 1998 - VML proposed by Microsoft, PGML proposed by IBM, Adobe, Netscape and sun - SVG working group was formed. 2001 - Adobe released the SVG Viewer 3 2001 - W3C makes a recommendation (41% of Americans were on dial-up. 6% on broadband) internet users were still on dial-up) http://www.pewinternet.org/2013/08/26/home-broadband-2013/ 2005 - Adobe adopts Flash 2006 - Opera 9 is the first browser with native support 2008 - Apple blocks Flash from iOS devices 2009 - Safari, Firefox then Chrome add support 2011 - IE9 is released, marking the last major browser to support 2012 - Apple releases retina devices 2012 - Microsoft drops support for VML with the release of IE10

#horizontal,
#stacked {
  display: none;
}

/* Horizontal */
@media all and (min-aspect-ratio: 570/270) {
  #horizontal { display: block; }
  #mark { display: none; }
}

/* Stacked */
@media all and (max-aspect-ratio: 12/18) {
  #mark,
  #stacked-design { display: none; }
  #stacked { display: block; }
}

/* Full */

  #stacked-design { display: block; }
